PACMAN GAME by Artiom Ovechko

===============================
GENERAL INFO
===============================
PacMan(r) implementation in WPF
compatible with plugins, which
you can write implementing 
GhostBehavior abstract class 
of source project's code.

===============================
REQUIREMENTS
===============================
In order to run the game .Net 
framework 4.5.2 or higher needs 
to be installed on your system.

==============================
INSTALLATION
==============================
Game core folder is compressed 
in PacManGame.zip file. Just
extract it wherever you want
and run GameView.exe

==============================
PLUGINS
==============================
In order to implement your own
behavior of ghost you need to
use Ghost.dll in Ghost/Bin/Debug.
